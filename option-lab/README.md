# Option Situation Lab

Static GitHub Pages app for scanning option-chain JSON snapshots into multi-leg debit combinations.

Current dataset: SPCX Dec 15 2028 option chain manually extracted from IBKR mobile screenshots while the market was closed.

Important: this is a research/education tool, not a live quote or order-routing tool. Always confirm combo bid/mid/ask in IBKR Strategy Builder before placing any order.
